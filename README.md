# Promptwatch Analytics - GTM Template

Google Tag Manager template for [Promptwatch Visitor Analytics](https://promptwatch.com) - Track your AI search visibility.

## Installation

### Community Template Gallery

1. Open your GTM container
2. Go to **Templates** > **Tag Templates** > **Search Gallery**
3. Search for "Promptwatch Visitor Analytics"
4. Click **Add to workspace**
